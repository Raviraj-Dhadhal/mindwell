# 🧠 MindWell Platform

**Your Mental Health & Wellness Hub**

MindWell is a comprehensive, student-focused mental health platform designed to help navigate academic stress while maintaining emotional well-on. It features secure mood tracking, academic resource libraries, and administrative support systems.

---

### 🌟 Key Features
*   **📊 Student Dashboard:** Track your daily mood, stress levels, and emotional trends.
*   **🛡️ Admin Oversight:** Advanced monitoring for moderators to provide personalized support to students.
*   **📚 Academic Resources:** Targeted help for anxiety, productivity, and mental wellness.
*   **💬 Support System:** Direct communication between admins and students.
*   **👤 Role Management:** Secure login and redirect system for Students vs. Admins.

---

### 🛠️ Technology Stack
*   **Frontend:** Vanilla JS, Modern CSS (Responsive), HTML5.
*   **Backend:** Express.js (Node.js).
*   **Database:** SQLite (Better-SQLite3) for fast, secure storage.
*   **Architecture:** Modular REST API with Token-based Authentication.

---

### 🚀 Getting Started (Local)
1.  Clone the repository: `git clone https://github.com/[YOUR_USERNAME]/MindWell.git`
2.  Install dependencies: `npm install`
3.  Start the server: `node server/index.js`
4.  Open: `http://localhost:3000`
